### サービス概要
Railsと楽天トラベルキーワード検索APIを連携させ作成したLINEbotです。  
送信されたキーワードに合致するホテルを複数件表示します。
場所だけでなく、「ビューバス」などのこだわりのワードでも検索することができます。

### 使用したAPI
- LINE Messaging API
- 楽天トラベルキーワード検索API
